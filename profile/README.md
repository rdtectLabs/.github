# rdtect Labs

**Exploration · Experiments · Creative Code**

The playground for [@rdtect](https://github.com/rdtect). Forks, experiments, generative art, OSS compositions, and fire-and-forget deployments.

Production work lives at [rdtect](https://github.com/rdtect). This is where ideas get tested before they earn their place.

---

### What's Here

| Type | Examples |
|------|----------|
| **Creative code** | Generative art, WebGL experiments, animation libraries |
| **Deployed sites** | Client projects and event pages (some archived, still live on Cloudflare) |
| **Forks & experiments** | OSS explorations, learning repos, proof-of-concepts |
| **Tools** | Project flow analyzers, wellness trackers, utility apps |

### Philosophy

> "The best abstraction is deletable." — MRAX Framework

Labs repos don't need to be polished. They need to exist, teach something, and either graduate to production or stay as reference.
